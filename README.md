## Preparation
Install virtual environment for Python packages (Linux)
```bash
python -m venv virt_env
source virt_env/bin/activate
```
oder in Windows:
```bash
python -m venv virt_env
.\virt_env\Scripts\activate
```


And install the following packages (see docu at https://github.com/oberth-effect/iodd-parser)

```bash
pip install iodd-parser
pip install python-dotenv
```
